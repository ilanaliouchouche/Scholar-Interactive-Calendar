	Identifiant : Etudiant
	mot de passe : etudiant

Compte coordinateur : 
	Identifiant : Coordinateur
	mot de passe : coordinateur

Compte Responsable :
	Identifiant : Responsable
	mot de passe : responsable


sinon vous pouvez créer un compte avec vos propres identifiants et le rôle désiré.
